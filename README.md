# Taýpa Telegram Post Bot

Бот принимает фотографию, определяет тип вещи, придумывает название принта через Gemini API, предлагает выбрать размеры, спрашивает цену, показывает предпросмотр и публикует пост в Telegram-канале сразу или в выбранное время.

## Возможности

- Доступ только для владельца по Telegram ID.
- Автоматическое определение вещи: футболка, худи, свитшот, лонгслив или кепка.
- Короткое подходящее название принта в кавычках.
- Два хэштега: один по типу вещи, второй по тематике принта.
- Выбор размеров кнопками: S-XXL, XS-4XL, XS-3XL или свой вариант.
- Запрос цены после анализа фотографии.
- Быстрый выбор времени или ввод точной даты.
- Предпросмотр перед подтверждением.
- Кнопка «Написать» открывает личный чат и заранее вставляет название товара.
- Изменение текста, цены и времени до публикации.
- Список запланированных постов и отмена публикации.
- Хранение очереди в SQLite, поэтому перезапуск не удаляет задания.
- До 5 автоматических попыток при временной ошибке Telegram.
- Редактируемый шаблон подписи.

## Быстрый запуск на Windows

### 1. Создайте бота

1. Откройте в Telegram `@BotFather`.
2. Отправьте команду `/newbot`.
3. Укажите название и username нового бота.
4. Скопируйте выданный токен. Он нужен для `TELEGRAM_BOT_TOKEN`.

### 2. Подготовьте канал

1. Откройте настройки канала.
2. Перейдите в раздел администраторов.
3. Добавьте созданного бота.
4. Включите ему право публикации сообщений.
5. Если канал публичный, скопируйте его username вместе со знаком `@`. Он нужен для `CHANNEL_ID`.

### 3. Узнайте ADMIN_TELEGRAM_ID

Это ваш личный цифровой ID. Это не номер телефона и не username.

1. Откройте в Telegram бота `@userinfobot`.
2. Нажмите `Start`.
3. Найдите строку `Id`, например `Id: 726543210`.
4. В `.env` запишите только цифры: `ADMIN_TELEGRAM_ID=726543210`.

Этот ID нужен, чтобы управлять ботом могли только вы.

### 4. Укажите CHANNEL_ID

Если канал публичный и его ссылка выглядит как `t.me/taypa_print`, используйте:

```env
CHANNEL_ID=@taypa_print
```

Если канал закрытый и username отсутствует, нужен цифровой ID вида `-1001234567890`:

1. Добавьте своего бота администратором канала.
2. Опубликуйте в канале любое тестовое сообщение.
3. Перешлите его боту `@RawDataBot`.
4. В ответе найдите блок `chat` и поле `id`.
5. Скопируйте полное число, начинающееся с `-100`.

### 5. Укажите CONTACT_USERNAME

Это ваш личный username, на который ведет кнопка «Написать».

1. Откройте Telegram, затем `Настройки` и `Имя пользователя`.
2. Если username еще нет, создайте его.
3. Укажите его без знака `@`.

Например, для ссылки `t.me/taypa_manager` используйте:

```env
CONTACT_USERNAME=taypa_manager
```

### 6. Создайте ключ Gemini API

1. Откройте `https://aistudio.google.com/app/apikey`.
2. Войдите в Google-аккаунт.
3. Нажмите `Create API key`.
4. Скопируйте ключ. Он нужен для `GEMINI_API_KEY`.
5. Никому не отправляйте этот ключ.

### 7. Установите и запустите

1. Установите Python 3.11 или новее с сайта `python.org`. При установке включите пункт `Add Python to PATH`.
2. Запустите `setup.bat` и дождитесь окончания установки.
3. Откройте созданный файл `.env` в Блокноте.
4. Замените примеры своими данными и сохраните файл.
5. Запустите `run.bat`. Не закрывайте это окно, пока бот должен работать.
6. Откройте бота в Telegram и отправьте `/start`.
7. Отправьте `/check`, чтобы проверить доступ к каналу.

Если ранее появлялась ошибка `неизвестный часовой пояс Asia/Ashgabat`, снова запустите обновленный `setup.bat`. Он установит базу часовых поясов для Windows. В боте также предусмотрен запасной режим UTC+5.

## Настройка .env

```env
TELEGRAM_BOT_TOKEN=токен_от_BotFather
GEMINI_API_KEY=ключ_Gemini_из_Google_AI_Studio
ADMIN_TELEGRAM_ID=ваш_цифровой_Telegram_ID
CHANNEL_ID=@имя_канала
CONTACT_USERNAME=ваш_username_без_знака_собака
TIMEZONE=Asia/Ashgabat
GEMINI_MODEL=gemini-3.1-flash-lite
BUTTON_TEXT=Написать
COPY_LANGUAGE=ru
DATABASE_PATH=data/posts.sqlite3
CAPTION_TEMPLATE_PATH=caption_template.txt
```

Для закрытого канала в `CHANNEL_ID` укажите числовой ID вида `-1001234567890`.

Если в перенесенном из старой версии `.env` осталась модель
`gemini-2.5-flash`, бот автоматически заменит ее на актуальную модель.

## Команды

- `/start` - открыть главное меню.
- `/queue` - показать запланированные посты.
- `/template` - показать шаблон подписи.
- `/settemplate` - изменить шаблон прямо в Telegram.
- `/check` - проверить доступ бота к каналу.
- `/cancel` - отменить создание текущего черновика.

## Как опубликовать пост

1. Отправьте боту фотографию.
2. Подождите, пока он определит вещь и создаст название принта.
3. Выберите размеры или впишите свой вариант.
4. Отправьте цену. Если написать только `250`, бот поставит `250 манат`.
5. Выберите быструю дату или нажмите «Другая дата и время».
6. Проверьте фотографию, подпись, цену, размеры, время и кнопку в предпросмотре.
7. При необходимости измените название, размеры, цену или время.
8. Нажмите «Подтвердить».
9. Для просмотра и отмены будущих публикаций используйте `/queue`.

Чтобы пост вышел в заданное время, программа должна быть запущена. Для круглосуточной работы используйте VPS и Docker.

Никому не отправляйте файл `.env`, токен Telegram и ключ Gemini. Если секрет попал к постороннему, сразу замените его.

## Шаблон подписи

Файл `caption_template.txt` можно менять вручную или командой `/settemplate`. Доступные русские поля:

- `{Тип товара}` - футболка, худи, свитшот, лонгслив или кепка.
- `{Название принта}` - название, созданное по изображению.
- `{Короткое описание, передающее настроение принта}` - короткая фраза для товара.
- `{Размеры}` - выбранные размеры.
- `{Цена}` - цена, которую вы отправили боту.
- `{тип товара}` - тип товара строчными буквами для хэштега.
- `{тематика принта}` - тематический хэштег без знака `#`.

Старые поля `{title}`, `{description}`, `{size}`, `{price}` и `{hashtags}` также поддерживаются.

Стандартная подпись выглядит так:

```text
Футболка "Welcome to Turkmenistan"

Футболка, которая передает атмосферу нашей страны ❤️

👕 S-XXL

💸 250 манат

⏳ Готовность заказа: 1-2 дня

Чтобы заказать, отправьте:

✅ размер
✅ цвет
✅ номер телефона

📩 Заказы принимаем в сообщениях 👇

#Taýpa #футболка #туркменистан
```

Telegram разрешает до 1024 символов в подписи к фотографии. Бот проверяет это до публикации.

## Постоянная работа 24/7

Запланированные публикации отправляются только пока бот запущен. Для постоянной работы разместите проект на VPS с Docker:

```bash
cp .env.example .env
nano .env
docker compose up -d --build
```

Проверка журнала:

```bash
docker compose logs -f bot
```

Остановка:

```bash
docker compose down
```

## Проверка

После установки зависимостей:

```bash
python -m unittest discover -s tests -v
```
